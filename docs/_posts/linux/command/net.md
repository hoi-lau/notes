# 网络命令

## write

w查看在线用户

给用户(登陆状态)发信息,CTRL+D保存结束

## wall

write all

给所有在线用户发信息

## ifconfig

yum install -y net-tools

## mail

yum install mailx sendmail

发送邮件

## last

查看过去所有登陆信息

lastlog 查看最近访问时间

## traceroute

显示数据包到主机间的路径

## netstat

显示网络相关信息

- -t tcp
- -u udp
- -l listening
- -r route
- -n ip port
